# 井字棋"段位"挑战

一个基于 Web 的井字棋（Tic-Tac-Toe）教学挑战游戏，玩家需要逐级挑战不同难度的 AI，通过段位系统评估技能水平。

## 项目简介

这是一个用于教学的单页面井字棋游戏，通过渐进式难度挑战帮助玩家理解 AI 决策过程和算法复杂度。游戏采用段位制，玩家需要从难度 0 开始，逐步挑战更高难度的 AI。

## 功能特点

- 🎮 **渐进式难度系统**：从难度 0 到难度 5，逐级挑战
- ⏱️ **倒计时机制**：每回合 30 秒倒计时，培养快速决策能力
- 🤖 **智能 AI 对手**：使用 Minimax 算法，根据难度调整搜索深度
- 📊 **实时思考时间显示**：观察 AI 在不同难度下的思考时间变化
- 🎯 **段位评价系统**：根据最高挑战难度评定段位
- 💡 **机会机制**：每局有 3 次失败机会，用完后显示最终段位

## 游戏规则

1. **玩家先手**：玩家使用 X，AI 使用 O
2. **难度提升**：每局获胜后，难度自动提升 1 级
3. **失败机制**：每局有 3 次失败机会（包括平局）
4. **时间限制**：玩家每回合有 30 秒思考时间
5. **段位评定**：3 次机会用完后，根据当前难度显示段位

## 技术实现

### 核心算法

- **Minimax 算法**：AI 使用 Minimax 算法进行决策
- **深度限制**：根据难度级别（0-5）调整搜索深度
  - 难度 0：随机落子
  - 难度 1-5：Minimax 搜索深度对应 1-5 层
- **计算模拟**：通过模拟计算负载，让不同难度的思考时间差异更明显

### 技术栈

- **纯 HTML/CSS/JavaScript**：单文件实现，无需额外依赖
- **响应式设计**：现代化的 UI 设计，良好的用户体验
- **实时更新**：动态显示游戏状态、倒计时、AI 思考时间等

## 游戏界面

- **游戏信息面板**：显示当前难度和剩余机会
- **状态提示**：实时显示游戏状态和回合信息
- **倒计时器**：30 秒倒计时，最后 10 秒会有警告提示
- **AI 思考时间**：显示 AI 每次决策的思考时间
- **段位显示**：挑战结束后显示最终段位

## 教学用途

本游戏设计用于教学场景，可以帮助学生：

1. **理解算法复杂度**：通过观察不同难度下 AI 的思考时间，理解搜索深度对计算量的影响
2. **体验 AI 决策过程**：了解 Minimax 算法的工作原理
3. **培养策略思维**：在时间压力下做出最优决策
